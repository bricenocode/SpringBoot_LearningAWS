## 🗺️ Learning Roadmap

The project covers the essential pillars of the AWS ecosystem:

### 1. 🏗️ Computing & Orchestration
* Deployment strategies using **Amazon EC2**, **AWS Fargate (ECS)**, and **AWS Lambda**.
* Containerization with **Docker** and **Amazon ECR**.

### 2. 🗄️ Database Management
* Relational database integration with **Amazon RDS** (PostgreSQL/MySQL).
* NoSQL implementations using **Amazon DynamoDB**.

### 3. 📨 Integration & Messaging
* Decoupling microservices with **Amazon SQS** (Queues) and **Amazon SNS** (Pub/Sub).
* Implementing event-driven patterns.

### 4. 🌐 Networking & API Exposure
* Designing **VPC** layouts (Public/Private subnets, NAT Gateways).
* Exposing services via **Application Load Balancer (ALB)** and **Amazon API Gateway**.

### 5. 🔐 Security & Configuration
* Centralized configuration with **AWS Systems Manager (Parameter Store)**.
* Sensitive data management using **AWS Secrets Manager**.
* Fine-grained access control with **AWS IAM**.

### 6. 📁 Storage & Static Content
* Object storage with **Amazon S3**.
* Content delivery and caching via **Amazon CloudFront**.

### 7. 📈 Observability & Metrics
* Centralized logging and monitoring with **Amazon CloudWatch**.
* Distributed tracing and performance analysis with **AWS X-Ray**.
